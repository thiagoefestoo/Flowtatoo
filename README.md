# Flowtatoo

Aplicativo mobile para o dono de um estúdio de tatuagem gerenciar agenda, clientes, pagamentos, indicadores e alertas.

## Estrutura

- `frontend`: React + Vite
- `backend`: Node.js + Express + Sequelize
- banco: PostgreSQL/Neon

Consulte `README_DEPLOY.md` para publicação e `README_OTIMIZACAO.md` para as mudanças de desempenho desta versão.
